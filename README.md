# Grand Brother - Face Detection System

Computer Vision project for educational robotics (PhD thesis).

## Status: In Development
- [x] Hardware setup
- [ ] YOLOv8 detection
- [ ] Face recognition
- [ ] Multi-camera support

## Hardware
- Raspberry Pi 5 (8GB)
- Debian GNU/Linux 13 (trixie)
- Python 3.13.5

## Setup
```bash
# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```
